# Description

Ce dépôt contient l'exercice et le projet que j’ai réalisé lors de la formation en ligne organisée par **AfriAI Solution** sur LinkedIn, les 13 et 14 janvier 2026, portant sur **"L'infrastructure pour faire de l'IA et du développement logiciel"**.

---

# Objectifs de la formation

La formation s’est principalement concentrée sur l’utilisation des outils suivants :

## Visual Studio Code
- Installation et configuration d’un environnement de développement indépendant.
- Mise en place d’un environnement de travail propre et structuré.

## Git & GitHub
- Utilisation des commandes principales de Git (`init`, `add`, `commit`, `push`, `pull`, `branch`, etc.).
- Gestion des versions du projet.
- Utilisation du fichier `.gitignore`.
- Publication et gestion d’un dépôt sur GitHub.

## Docker
- Introduction à la conteneurisation.
- Création d’une petite application.
- Création d’un `Dockerfile`.
- Construction et exécution d’une image Docker pour déployer l’application.
